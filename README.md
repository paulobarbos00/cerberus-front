# Cerberus - Front

### O Projeto Cerberus consiste em um serviço de organização e catálogo de links para auxiliar o usuário a acessar seus conteúdos de forma prática.


## Tecnologias:
- React
- NextJS v13
- TypeScript
- ESLint
- Prettier
